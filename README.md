# Davidious AI Code Lab Trainer

## AI Trainer for Coding, Tracking and Evaluating progress

### AI coding lab with task generation, sandbox code execution, evaluation, progress tracking, and timed interview simulation. 

# Tech Stack

- Python
- FastAPI
- OpenAI API
- LangChain
- Typescript
- React
- PostgreSQL
- SQLModel ORM
- Docker

---

# Features

- AI-driven coding task generation 
- Dockerized secure sandbox code execution 
- Automated solution evaluation system 
- Timed interview simulation module 
- Documents and youtube course integration  
- Email reminders to practise weak skills 
- Progress tracking and analytics
